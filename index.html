學長經驗分享

2016/09/06
Compressive Large-Scale Image Sensing

In this talk, we present an algorithm to solve convex optimization via the tree structure sparsity pattern, which can be run in the operator to reduce computation cost and maintain good quality, especially for large-scale images. The feasibility of our method is verified via computational complexity and convergence analyses, extensive simulations, and comparisons with state-of-the-art algorithms.
